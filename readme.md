## Requisitos:
- `NodeJS`
- `MariaDB ou MySQL`

## Como executar:
```shell
# Clone o projeto
$ git clone git@github.com:Throyer/nodejs-typescript-typeorm-routing-controllers.git nodejs-crud && cd nodejs-crud

# Baixe as dependencias
$ yarn install

# Execute o servidor node
$ yarn start
```
